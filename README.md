# Pokémon Brilliant Diamond Interactive Website

An interactive Pokémon Brilliant Diamond themed fan website built using HTML, CSS, and JavaScript.

This project recreates the world of Sinnoh through a modern web experience featuring animated UI components, interactive gameplay elements, character showcases, Pokémon information systems, and immersive design.

## Features

### Interactive Navigation

* Smooth scrolling navigation
* Dynamic active section highlighting
* Responsive dropdown menus

### Character System

* Interactive player character cards
* Editable character names
* Dynamic profile selection

### Starter Pokémon Explorer

* Turtwig, Chimchar, and Piplup information panels
* Dynamic content updates using JavaScript
* Interactive Pokémon selection

### Super Contest Mini Game

* Rhythm-based gameplay system
* Real-time score tracking
* Interactive note timing mechanics

### Sinnoh Region Showcase

* Gym Leaders
* Elite Four
* Champion Cynthia
* Team Galactic hierarchy
* Legendary Pokémon section

### Modern UI/UX

* Glassmorphism-inspired design
* Smooth animations and transitions
* Scroll-triggered effects using Intersection Observer API
* Responsive layouts for multiple screen sizes

### Pre-Order Experience

* Pokémon Brilliant Diamond themed pre-order modal
* Interactive checkout workflow
* Animated button effects

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* DOM Manipulation
* Intersection Observer API
* Responsive Web Design
